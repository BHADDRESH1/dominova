
# Dominova - Internal Management System

## Project Overview
Dominova is a comprehensive internal management system designed to streamline HR, Ambassador programs, and administrative tasks.

## Tech Stack
- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Supabase (Backend & Auth)

## Getting Started

### Prerequisites
- Node.js & npm installed

### Installation

1. Clone the repository
```sh
git clone <YOUR_GIT_URL>
```

2. Install dependencies
```sh
npm install
```

3. Start the development server
```sh
npm run dev
```

## Features
- **Role-Based Access Control**: Separate dashboards for Owners, Admins, Ambassadors, and Employees.
- **Ambassador Management**: Track referrals, earnings, and leaderboard.
- **Employee Management**: Manage staff directory and profiles.
- **Verifications**: Admin tools for verifying tasks and payments.
- **Offer Letters**: Automated generation of offer letters.
